CSS

   CSS (Cascading Style Sheets) is a styling language used to describe the presentation of a document written in HTML. It allows web developers to control the layout, colors, fonts, and other visual aspects of web pages.

   Selector: A selector is used to target specific HTML elements that you want to style.

   Declaration: A declaration consists of a property and a value, defining how the selected elements    should be styled.

   Property: A property determines the specific aspect of an element that you want to style.

   Value: A value is assigned to a property and defines how the selected elements should appear.

Commonly Used CSS Properties


  1. Color

    Specifies the text color of an element.

    Example: color: red;

  2. Font-Size

    Specifies the size of the text within an element.

    Example: font-size: 16px;

  3. Background-Color

    Specifies the background color of an element.

    Example: background-color: #f5f5f5;

  4. Margin

    Specifies the margin area around an element.

    Example: margin: 10px;

  5. Padding

    Specifies the padding area around the content of an element.

    Example: padding: 10px;

  6. Border

    Specifies the border properties of an element.

    Example: border: 1px solid #ddd;
